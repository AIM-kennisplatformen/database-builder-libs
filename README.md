# Database Builder Libs

A Python library for building knowledge graph applications with TypeDB and Qdrant.

## Overview

Database Builder Libs provides a unified interface for:
- Extracting data from various sources (Zotero, files, APIs)
- Storing knowledge in graph databases (TypeDB)
- Creating vector embeddings for semantic search (Qdrant)

## Installation

Install directly from GitHub using pixi:

```bash
pixi add "git+https://github.com/AIM-kennisplatformen/database-builder-libs.git"
```
