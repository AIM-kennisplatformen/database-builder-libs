The database-builder-libs repository¶

Repository layout¶

The repository database-builder-libs is hosted on GitHub and contains the following layout: 

.
├── docs
├── LICENSE
├── mkdocs.yml
├── pixi.lock
├── pyproject.toml
├── README.md
├── src
└── tests

Branches¶

The core branch of database-builder-libs is the main branch. This branch contains the latest reviewed and tested release.

The xx-<feature_name> branches are created from the project-board and are develop branches which get merged into main and deleted.

Last but not least <feature_name> branches without number-prefix are hotfix branches, which get merged into main and deleted afterwards.
